# Regenerate Non-traumatic SAH Manuscript Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Regenerate high-quality English and Chinese manuscripts and PDFs in the `dist/20260707` directory based on the latest BigQuery and Python analysis results, and create a metadata readme file.

**Architecture:** Draft structured manuscripts in Markdown (`dist/20260707/manuscript_non_traumatic_sah_phenotypes.md` and `dist/20260707/manuscript_non_traumatic_sah_phenotypes_cn.md`) incorporating latest figures and statistics from `dist/20260707/analysis_result.md` and the eICU external validation results documented in `docs/eicu_external_validation.md`. Convert both manuscripts to PDF using the existing Python conversion script `scripts/convert_manuscript_to_pdf.py`, and write a metadata readme file `dist/20260707/readme.txt`.

**Tech Stack:** Python 3, Markdown, WeasyPrint, matplotlib.

## Global Constraints

- Follow the guidelines in `AGENTS.md` and `docs/high_impact_manuscript_standards.md` strictly.
- All numbers must be 100% accurate and aligned with the latest values in `dist/20260707/analysis_result.md` and `docs/eicu_external_validation.md`.
- No placeholders or TBD markers are allowed in the final manuscripts.
- Avoid over-stating clinical findings or asserting causal treatment effects (particularly regarding early anemia and process-of-care variables).
- All figures must be referenced at their appropriate locations in the text (e.g. `**Figure X.** Legend...`).
- The title of the English manuscript must be: "Early Multimodal Physiological Phenotypes and Outcomes in Critically Ill Adults With Non-Traumatic Subarachnoid Hemorrhage: A Retrospective Cohort Study Using MIMIC-IV 3.1".
- All output files must be saved under `dist/20260707/`.
